# VisualBricks Designer

> 通用无代码设计器
>
>

## ✨ 特性

- 源自实际业务的无代码解决方案，开源、免费
- 专业级别的设计工具套件
- 图灵完备的图形化编程语言及扩展支持
- 强大且开放的自定义组件能力

## 🖥 兼容环境

- 现代浏览器（包括IE11）

|Chrome|IE / Edge | Firefox  |Safari | 
| --- | --- | --- | --- |
| last 2 versions | IE11, Edge | last 2 versions | last 2 versions |

## 📦 安装及体验

```bash
npm install @visualbricks/designer --save
npm run dev
```

```bash
yarn add @visualbricks/designer
yarn run dev
```
