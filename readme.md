# VisualBricks Designer

> VisualBricks出品的通用无代码设计器
> 
>

## ✨ 特性
- 面向实际业务的无代码解决方案
- 专业级别的设计工具套件
- 图形化编程语言（图灵完备）
- 强大且开放的自定义组件能力


## 🖥 兼容环境

- 现代浏览器和 IE11（需要 [polyfills](https://ant.design/docs/react/getting-started-cn#兼容性)）。
- 支持服务端渲染。
- [Electron](https://www.electronjs.org/)
